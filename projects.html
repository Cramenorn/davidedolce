Law Cloud

Diego Robaldo, Andrea Tallone, Davide Dolce, Simone Filippo Antonicchio, Mariangela Pastore, Pietro Giovanetti, Simone Russo, Francesco Suriano

A project made in team with other school mates, LawCloud is an app mainly designed for lawyers, they can chat with their customers, create events, create folders to upload documents, create a cause and more. We developed two versions, one for IOS and another one for Android platform. I was responsible for the creation of the BackEnd API made with PHP, Laravel and OAuth.

RaptorSA

Davide Dolce

RaptorSA is a multi function bot, with more than 40 commands, you can listen music, manage roles, get various informations and many other things. This Discord bot can be helpful for anyone who wants to learn how to make a discord.js bot. I made this so new users can also look at the methods used for certain commands and get some ideas for their Discord bot.

View the source code on Github
